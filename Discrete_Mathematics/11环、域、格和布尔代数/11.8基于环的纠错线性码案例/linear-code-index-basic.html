(3,2) 偶校验线性码

蓝色=信息位，绿色=校验位，红色=出错位。G = [I₂ | P]，校验位 p = u₁ ⊕ u₂。

📡 线性码 = 有限域上的向量空间

全部码字构成 F₂ 上的向量空间：含零码字、对加法 ⊕ 封闭（任两码字之和仍是码字）——这正是『线性』的含义，也是加法子群。线性结构让编码 c=u·G 与校验都化为矩阵运算。本例 d=2，只能检测 1 位错误；要定位纠正需更大的最小距离（进阶层）。
