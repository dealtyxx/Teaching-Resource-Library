🎯 布尔代数的定义

布尔代数是一个代数系统 B = (B, +, ·, ¬, 0, 1)，其中：

• B：布尔集合，通常为 {0, 1}

• +：逻辑或运算（并）

• ·：逻辑与运算（交）

• ¬：逻辑非运算（补）

• 0：零元（假）

• 1：幺元（真）

📚 布尔代数基本定律

🔹 交换律 (Commutative Law)
A + B = B + A
A · B = B · A
🔹 结合律 (Associative Law)
(A + B) + C = A + (B + C)
(A · B) · C = A · (B · C)
🔹 分配律 (Distributive Law)
A · (B + C) = (A · B) + (A · C)
A + (B · C) = (A + B) · (A + C)
🔹 同一律 (Identity Law)
A + 0 = A
A · 1 = A
🔹 零一律 (Null and Identity Law)
A + 1 = 1
A · 0 = 0
🔹 互补律 (Complement Law)
A + ¬A = 1
A · ¬A = 0
🔹 幂等律 (Idempotent Law)
A + A = A
A · A = A
🔹 吸收律 (Absorption Law)
A + (A · B) = A
A · (A + B) = A
🔹 德摩根定律 (De Morgan's Law)
¬(A + B) = ¬A · ¬B
¬(A · B) = ¬A + ¬B
🔹 双重否定律 (Double Negation Law)
¬(¬A) = A
