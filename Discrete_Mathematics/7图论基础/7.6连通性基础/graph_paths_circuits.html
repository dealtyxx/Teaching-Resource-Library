通路是图中从一个顶点到另一个顶点的路径,如同革命征程。

当前概念: 通路
路径长度: -

图论核心概念

通路 (Path)

→

初级通路

除起点和终点外,其余顶点不重复的通路

v₁ → v₂ → v₃ → ... → vₙ (顶点不重复)

例: 长征从江西瑞金到陕北延安

⇒

简单通路

边不重复的通路

e₁, e₂, e₃, ..., eₙ (边不重复)

例: 古代丝绸之路贸易线

回路 (Circuit)

↻

初级回路

起点和终点相同,其余顶点不重复的通路

v₁ → v₂ → ... → vₙ → v₁ (形成循环)

例: 革命根据地巡视路线

⟲

简单回路

起点和终点相同,边不重复的回路

闭合路径,边各异

例: 经济循环系统

连通分量 (Connected Component)

◎

最大连通子图

图中的极大连通子图,任意两点间存在通路

∀u,v ∈ V, ∃ path(u,v)

例: 革命根据地连成一片

✓ 内部完全连通
✓ 不可再扩展
✓ 分量间不连通
